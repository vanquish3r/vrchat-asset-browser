# VRChat Asset Database

A simple, fast, and searchable website to browse thousands of VRChat assets, tools, prefabs, shaders, and tutorials, all consolidated from community-driven spreadsheets.

This project indexes and provides an easy-to-use interface for the data compiled by the VRChat community.

**[View the Live Website Here](https://vanquish3r.github.io/vrchat-asset-browser/)**

## Features

* **Fast Search:** Instantly search across 700+ assets by name, author, or description.
* **Filter & Sort:** Easily filter by main categories (like `Prefabs`, `Shaders`, `Tools`) and various sub-categories.
* **Direct Links:** Get direct links to download or view the asset from the original source (Gumroad, Booth, GitHub, etc.).
* **Dark/Light Mode:** Automatically respects your system's theme and provides a toggle to switch.
* **Fully Static:** The site is built with plain HTML, CSS, and JavaScript. It's incredibly fast and easy to host for free on GitHub Pages.

## Data Sources

This database is a compilation of data from the following community-maintained Google Sheets:

* [VRC Prefabs](https://vrcprefabs.com/)

The HTML, CSS, and JavaScript code for this website was generated by Google's Gemini, an AI assistant, based on prompts provided by a human user. The AI processed and structured the data from the provided CSV files. The content itself (the asset descriptions, author names, and links) originates from the community-compiled spreadsheets and is presented as-is.
