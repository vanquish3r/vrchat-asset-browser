# VRChat Asset Database

A simple, fast, and searchable website to browse hundreds of VRChat assets, tools, prefabs, shaders, and tutorials. This project indexes and provides an easy-to-use interface for data compiled by the VRChat community.

**[View the Live Website Here](https://vanquish3r.github.io/vrchat-asset-browser/)**

## Features

* **Fast Search:** Instantly search across all assets by name, author, or description.
* **Filter & Sort:** Easily filter by category (like `Prefabs`, `Shaders`, `Tools`) and sort by name or submission date.
* **Detailed Info:** Each card displays primary info (Name, Author, Description) and secondary info (Submitted by, Date, Notes).
* **Direct & Preview Links:** Provides a direct "Download / View" link for the asset and a "Preview" button if a preview link is available.
* **Clickable Links:** Any URLs found in the "Notes" section are automatically made clickable.
* **Dark/Light Mode:** Includes a theme toggle with icons.
* **Fully Static:** Built with plain HTML, CSS, and JavaScript, making it fast and easy to host for free on GitHub Pages.
